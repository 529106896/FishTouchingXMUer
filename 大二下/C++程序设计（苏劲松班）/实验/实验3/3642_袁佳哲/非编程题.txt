课外习题：
1)
①指向常量的指针：也叫作常量指针，const修饰的是指针，指针的指向可以改变，指针指向的值不可以改变
如：
int a = 10;
int b = 10;

const int * p1 = &a;//p1是指向常量的指针
p1 = &b;        //指向可以改
//*p1 = 100;    //指向的值不可以改，会报错


②指针类型的常量：也叫作指针常量，const修饰的是常量，指针指向的值可以改变，指针的指向不可以改变
如：
int a = 10;
int b = 10;

int * const p2 = &a;
//p2 = &b;      //指向不可以改
*p2 = 100;      //指向的值可以改

③区分的方法：
一个最简单的功能区分方法是从定义上区分
如：	const int * p1; //直接看，p1指向const int，所以这是指向常量的指针，指向可以变，但指向的值不可以变
	int * const p2;//指针的名字是const p2，而const p2的类型是int *，说明是指针类型的常量，指向不可以变，指			向的值可以变

2)
动态变量比较容易区分，基本上都是用new、delete来操作的，动态变量的内存需要由程序员自己去划分与释放
非动态变量分为一般变量与静态变量，一般的变量就是平时很常见的直接一个int a; 前面没啥关键字（auto可以被被忽略），每次调用函数就会自动给他分配内存，函数执行完毕就自动释放；静态变量的关键字是static，它的生存期从它第一次被定义就开始了，一直到整个程序结束才会被释放

3)
动态变量的创建方法需要用到关键字new，比如
Node *head = new Node;
int *array = new int[n];
int *a = new int(10);

基本是每个new都要对应一个delete
delete p;
delete a;
delete[] array;
